# 01 — Health · 健康与精力

> 一切决策的物理底座。健康崩盘则现金流、求职、商业探索全部失效。

## CDO 视角

健康在 Axiom Core 优先级链中位于**最高位**——任何商业决策、学习投入、社交活动都必须让位于此。它不是"加分项"，而是**所有其他指标的乘数因子**：精力为零，再好的项目机会也变成压力放大器。

## 追踪的核心指标

| 指标 | 录入频率 | 红线 |
|------|---------|------|
| 睡眠时长（小时） | 日 | 连续 3 天 < 5h → survival 模式 |
| 睡眠质量（poor/fair/good） | 日 | 连续 3 天 poor → 暂停所有高强度学习/训练 |
| 体重（kg） | 周 | 周降 > 2kg → 医学告警 |
| 精力分数（1-10） | 日 | 7 天均值 < 4 → 暂停新项目 |
| 心情分数（1-10） | 日 | 7 天均值 < 4 → 复盘压力源 |
| 运动分钟 | 日 | 求职高峰期 ≤ 3h/周 |
| 体检指标 | 季度 | 任意红色指标 → 优先就医 |

## 录入规范

1. **数据先于解读**：先录原始数字，再写主观感受。不能用感受替代数据。
2. **不补录**：当天没记录就空着，第二天不要回填——历史数据失真比缺失更糟。
3. **异常优先**：胸痛、晕厥、异常出血、持续头痛 ≥ 3 天 → 立即触发决策引擎 §4.2 红线，不等周报。

## 与决策引擎的关系

决策引擎（[../docs/decision-engine.md](../docs/decision-engine.md) §4）从此目录读取信号：
- 进入 `survival` 模式的健康触发条件
- 拒绝高强度训练 / 高强度脑力工作的硬否决依据
- 紧急豁免（自伤念头、医学红灯）的判定来源
