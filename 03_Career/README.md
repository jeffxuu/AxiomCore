# 03 — Career · 事业与商业探索

> 商业探索与核心技能变现路径。**当前主线**：在 survival 模式下，求职恢复稳定现金流；中长期：独立商业探索。

## CDO 视角

Career 不等于"打工 / 求职"。它是**让你的技能与时间产生稳定现金流的所有路径**：
- 短期：求职拿到 18K-22K 稳定岗位（覆盖最低还款 + 基础开销）
- 中期：把现场管理 / 全栈开发 / AI Agent 编排能力包装为可变现的服务
- 长期：独立商业（见 `05_Projects/`）

CDO 在这里只回答一个问题：**当前阶段的最大杠杆是哪条路径？** 其他路径暂时全部冻结。

## 追踪的核心指标

| 指标 | 频率 | 用途 |
|------|------|------|
| `job_applications`（投递数） | 日 | survival 期间核心动作 |
| `interviews`（面试数） | 日 | 转化率监控 |
| 目标薪资区间 | 季 | 18K-22K（当前） |
| 简历版本与投递渠道 | 月 | 复盘哪种版本/渠道转化率高 |
| 商业探索机会（lead） | 不定期 | 评估是否进入 `05_Projects/` 沙盘 |

## 录入规范

1. **投递动作 = 提交了简历 + 收到了通知**。"打开 BOSS 直聘看一眼"不算。
2. **面试动作 = 实际有了对话**（电话或视频或线下）。"约了时间但还没发生"不算。
3. **拒绝 = 写明理由**：HR 已读不回也算（标记 ghost），明确拒信另算（标记 reject + 原因）。

## 与决策引擎的关系

[../docs/decision-engine.md](../docs/decision-engine.md) §5 目标克制原则与本目录强耦合：
- survival 模式下严禁"用学习 / 调研 / 复盘"替代真实投递动作
- 多线开战 ≤ 1（求职动作不算新项目，但同时启动副业算）
- "做副业" 不能替代当前求职
